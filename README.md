# redalert
notifies you on red alert based on city filter 

# install (cross platform)

`$ go get github.com/yaronsumel/redalert/cmd/redalert`

# usage

All alerts (no filter)

`$ redalert`

Specific city

`$ redalert -filter="רמת גן"`
